Tiny Meanings

🧁 What is an algorithm?

An algorithm is a recipe for solving a problem. Step 1, step 2, step 3.

🔎 What is brute force?

Brute force means trying everything until it works. It is simple, but can be slow.

⏱️ What is time complexity?

Time complexity asks: If the problem gets bigger, how many more steps will we need?

🎒 What is space complexity?

Space complexity asks: How much extra memory does our plan need?

Dramatic comparison

Long Walks and Magic Jumps

Brute force: O(n) means steps grow with the number of things.

Binary search: O(log n) means steps grow very slowly because we cut the problem in half.

Brute force grows like a long walk. Binary search grows like magic jumps.
