🐸 The story

A frog starts on stone 0. From stone i it can jump to i+1 or i+2. Each jump costs |height diff|. Minimise total tiredness.

what is the recurrence?

The DP recurrence in plain math:

cost(i) = min(cost(i-1)+|h[i]-h[i-1]|, cost(i-2)+|h[i]-h[i-2]|)

int cost(int i, int[] h) {
    if (i == 0) return 0;
    int oneStep = cost(i-1, h) + Math.abs(h[i]-h[i-1]);
    int twoStep = (i > 1) ? cost(i-2, h) + Math.abs(h[i]-h[i-2]) : Integer.MAX_VALUE;
    return Math.min(oneStep, twoStep);
}

int[] memo;
int cost(int i, int[] h) {
    if (i == 0) return 0;
    if (memo[i] != -1) return memo[i];
    int a = cost(i-1, h) + Math.abs(h[i]-h[i-1]);
    int b = (i > 1) ? cost(i-2, h) + Math.abs(h[i]-h[i-2]) : Integer.MAX_VALUE;
    return memo[i] = Math.min(a, b);
}

    dp[i] = Math.min(dp[i-1] + Math.abs(h[i]-h[i-1]),
                     dp[i-2] + Math.abs(h[i]-h[i-2]));

👀 see the Java code (step 4: space saver)
JAVA   space-optimised DP ⬆ tiny memory 
int prev2 = 0, prev1 = Math.abs(h[1]-h[0]);
for (int i = 2; i < n; i++) {
    int curr = Math.min(prev1 + Math.abs(h[i]-h[i-1]),
                        prev2 + Math.abs(h[i]-h[i-2]));
    prev2 = prev1; prev1 = curr;
}
return prev1;
