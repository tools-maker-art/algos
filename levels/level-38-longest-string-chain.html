Longest String Chain

World E LIS

🎮 Objective

Find the longest chain of words where each next word adds exactly one letter.

🧠 Problem (in tiny words)

A predecessor of W differs by removing exactly one character. Find longest chain.

🎭 Story

A vocabulary-builder game: each turn, add one letter to the previous word.

😵 Brute force (the painful way)

Enumerate chains.

🌱 From brute → recursion

Sort by length. dp[w] = 1 + max(dp[predecessor]).

🔁 Recursion

Map word → length.
