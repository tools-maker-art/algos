➕➖ Plus or Minus?

Target Sum

World C Knapsack

🎮 Objective

Assign + or − to each number so the resulting expression equals target.

🧠 Problem (in tiny words)

Count ways to assign signs so the expression equals target.

🎭 Story

A boss says: “Use these numbers, each with + or −. Hit the magic number T. How many ways?”

😵 Brute force (the painful way)

2ⁿ sign combinations.

🌱 From brute → recursion

Let P = sum of +-nums, N = sum of −-nums. P-N=T, P+N=S → P=(S+T)/2. So count subsets with sum P. ⇒ Level 16.

🔁 Recursion

Reduces to Count Subsets with Sum K.
