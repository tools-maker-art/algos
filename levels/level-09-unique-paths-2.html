Unique Paths II

World B Grid DP

🎮 Objective

Same grid — now some cells are blocked. Count paths that avoid the obstacles.

🧠 Problem (in tiny words)

Grid contains 0 (free) and 1 (obstacle). Count paths from (0,0) to (m-1,n-1) avoiding all 1s.

🎭 Story

The fox now meets fallen boulders. Skip those cells entirely.

😵 Brute force (the painful way)

Same recursion — but skip blocked cells.

🌱 From brute → recursion

If grid[i][j]==1, paths(i,j)=0. Else paths(i,j)=paths(i-1,j)+paths(i,j-1).

Conditional recurrence as above.
