Wildcard Matching

World D Strings

🎮 Objective

Match a string against a pattern with '?' (any one char) and '*' (any sequence).

🧠 Problem (in tiny words)

Return whether pattern p matches string s.

🎭 Story

A search box that lets users type wildcards. Match or not?

😵 Brute force (the painful way)

Recursion on (i,j) trying all '*' expansions.

🌱 From brute → recursion

dp(i,j): if p[j]=='?' or s[i]==p[j] → dp(i-1,j-1). If p[j]=='*' → dp(i-1,j) (consume) OR dp(i,j-1) (skip *).
