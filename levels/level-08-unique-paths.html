🦊 Fox in the Forest

Unique Paths

World B Grid DP

🎮 Objective

Count unique paths from the top-left to the bottom-right of an m×n grid moving only right or down.

🧠 Problem (in tiny words)

Grid m×n. From (0,0) you only move Right or Down. How many distinct paths reach (m-1,n-1)?

🎭 Story

A fox lives at (0,0). Berries are at (m-1,n-1). It can only walk Right or Down. How many berry-routes exist?

😵 Brute force (the painful way)

Try every path recursively. 2^(m+n).

🌱 From brute → recursion

paths(i,j) = paths(i-1,j) + paths(i,j-1). Two choices reach a cell.

🔁 Recursion

paths(i,j) = paths(i-1,j) + paths(i,j-1). Base: paths(0,j)=paths(i,0)=1.

# pseudo
paths(i,j) = paths(i-1,j) + paths(i,j-1). Base: paths(0,j)=paths(i,0)=1.

🧩 Memoization

Cache by (i,j).

🧱 Tabulation

Fill 2D dp grid row by row.

🚀 Space optimisation

Single row of size n. Update in place.
