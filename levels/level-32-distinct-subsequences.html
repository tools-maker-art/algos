🔍 The story

Count distinct subsequences of S equal to T.

what is the recurrence?

The DP recurrence in plain math:

dp(i,j) = dp(i-1,j) + (S[i]==T[j] ? dp(i-1,j-1) : 0)

cnt(i, j, s, t):
    if (j == 0) return 1
    if (i == 0) return 0
    skip = cnt(i-1, j, s, t)
    take = (s.charAt(i-1) == t.charAt(j-1)) if cnt(i-1, j-1, s, t) else 0
    return skip + take
