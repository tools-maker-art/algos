🎒 Pirate&rsquo;s Greedy Bag

0/1 Knapsack

World C Knapsack

🎮 Objective

Pack a bag of capacity W with items (value, weight) to maximise total value. Each item once.

🧠 Problem (in tiny words)

Maximise value subject to total weight ≤ W. Each item picked at most once.

🎭 Story

A pirate raids a treasury. Each item has value & weight. Bag fits W kg. What’s the best haul?

😵 Brute force (the painful way)

Try all subsets (2ⁿ).

🌱 From brute → recursion

State (i, w). Pick → val[i]+best(i-1,w-wt[i]); skip → best(i-1,w). Max of the two.

🔁 Recursion

As above. Base best(-1,*)=0.

# pseudo
As above. Base best(-1,*)=0.

🧩 Memoization

2D cache.

🧱 Tabulation

dp[i][w]; row-by-row.

🚀 Space optimisation

1D dp[w], iterate w high→low.

📊 Complexity

💡 Key takeaway

Pick/skip with weight bounds = textbook 0/1 Knapsack.
