🔪 Slice into Palindromes

Palindrome Partitioning II

World G Partition DP

🎮 Objective

Cut a string into palindromic pieces with the minimum number of cuts.

🧠 Problem (in tiny words)

Min cuts so every piece is a palindrome.

🎭 Story

A printer accepts only palindrome words. Slice the message into the fewest palindrome chunks.

😵 Brute force (the painful way)

Try every cut combination.

🌱 From brute → recursion

dp[i] = min cuts for s[0..i]; if s[j..i] palindrome → dp[i] = min(dp[i], 1+dp[j-1]).
