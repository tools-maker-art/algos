🔢 How Many Ways?

Count Subsets with Sum K

World C Knapsack

🎮 Objective

Count how many subsets of nums[] add to exactly K.

🧠 Problem (in tiny words)

Return the number of subsets summing to K.

🎭 Story

The treasure has weight K. How many distinct selections of pebbles sum exactly to K?

😵 Brute force (the painful way)

Enumerate subsets.

🌱 From brute → recursion

cnt(i,k) = cnt(i-1,k) + cnt(i-1,k-nums[i]). Replace OR with +.

🔁 Recursion

As above. Base cnt(0,0)=1.

# pseudo
As above. Base cnt(0,0)=1.

🧩 Memoization

Cache (i,k).

🧱 Tabulation

dp[k] += dp[k - nums[i]] for each item.

🚀 Space optimisation

O(K).
