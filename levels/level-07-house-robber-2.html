Level 07

🏘️ Robber on a Roundabout

House Robber II

World A Basics

🎮 Objective

Same robber — but the houses form a circle. The first and last are now neighbours.

🧠 Problem (in tiny words)

House Robber, but circular array. First and last are adjacent.

🎭 Story

The houses now sit on a roundabout. Robbing both house 0 and house n-1 trips a single shared alarm.

😵 Brute force (the painful way)

Try all subsets respecting the rule. Exponential.

🌱 From brute → recursion

Split into two normal House Robber problems: (a) houses 0..n-2 (b) houses 1..n-1. Take the max.

🔁 Recursion

ans = max( robLine(v[0..n-2]), robLine(v[1..n-1]) )

# pseudo
ans = max( robLine(v[0..n-2]), robLine(v[1..n-1]) )

🧩 Memoization

Memoise each linear sub-call as in Level 6.

🧱 Tabulation

Run the linear DP twice on two slices.

🚀 Space optimisation

O(1) per linear pass.
