🪙 Tiny Cashier

Coin Change (min coins)

World C Knapsack

🎮 Objective

Make amount A using the fewest coins from given denominations (unlimited supply).

🧠 Problem (in tiny words)

Return min number of coins summing to A, or -1 if impossible.

🎭 Story

A cashier with infinite coins of given denominations wants to give exact change with as few coins as possible.

😵 Brute force (the painful way)

Try every coin at every step (exponential).

🌱 From brute → recursion

min(A) = 1 + min over coins c with c≤A of min(A-c).

🔁 Recursion

As above. Base min(0)=0.

# pseudo
As above. Base min(0)=0.

🧩 Memoization

1D cache.

🧱 Tabulation

dp[A]; dp[0]=0; for a=1..A take min over coins.

🚀 Space optimisation

O(A).
