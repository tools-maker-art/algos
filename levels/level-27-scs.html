✂️ Stitch the Two Notes

Shortest Common Supersequence

World D Strings

🎮 Objective

Find the shortest string that contains both inputs as subsequences.

🧠 Problem (in tiny words)

Length = n + m − LCS(A,B). Optionally print one such SCS.

🎭 Story

Take both spy notes, stitch them so each appears in order. The shortest stitched story is the SCS.

😵 Brute force (the painful way)

Enumerate stitches.

🌱 From brute → recursion

Length first via LCS. To print: walk LCS table; on match output once, on skip output the corresponding char.
