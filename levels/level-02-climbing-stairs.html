Climbing Stairs

🪜 The story

You stand at the bottom of n stairs. Each move you can take 1 or 2 steps. How many distinct climbs reach the top?

what is the recurrence?

The DP recurrence in plain math:

ways(n) = ways(n-1) + ways(n-2)

👀 see the Java code (step 2: + memory)
JAVA   recursion + memoization ⬆ each tiny problem solved once 
int[] memo;
int ways(int n) {
    if (n <= 1) return 1;
    if (memo[n] != -1) return memo[n];
    return memo[n] = ways(n-1) + ways(n-2);
}

JAVA   bottom-up tabulation ⬆ no recursion · just a loop 
int[] dp = new int[n+1];
dp[0] = 1; dp[1] = 1;
for (int i = 2; i <= n; i++) dp[i] = dp[i-1] + dp[i-2];
return dp[n];

🪶 Space saver: keep only what you need

Each new answer needs only the last two. Two ints suffice.

step 1

👀 see the Java code (step 4: space saver)
JAVA   space-optimised DP ⬆ tiny memory 
int prev = 1, curr = 1;
for (int i = 2; i <= n; i++) {
    int next = prev + curr; prev = curr; curr = next;
}
return curr;
