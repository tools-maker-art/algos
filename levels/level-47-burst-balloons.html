PSEUDO   fill the scoreboard ⬆ no recursion · just a loop 
N = nums.length
a = new boxes
a[0] = 1; a[N+1] = 1
for i from 0 to before N:
    a[i+1] = nums[i]
M = N + 2
dp = new boxes
for len from 2 to before M:
for (i = 0; i + len < M; i++)
    j = i + len
    for k from i+1 to before j:
    dp[i][j] = max(dp[i][j], dp[i][k] + dp[k][j] + a[i]*a[k]*a[j])
return dp[0][M-1]
